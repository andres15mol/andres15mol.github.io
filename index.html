Proyecto

Saber Algo / KnowSomething Blog



Interactive Blog - Developed with Golang, PostgreSQL, and Bootstrap

This project showcases my web development skills, where I built an interactive blog from scratch using key technologies such as Golang in the backend, PostgreSQL as the database management system, and Bootstrap for the frontend.


Key Features:

  • Secure Authentication: I implemented a registration and authentication system based on sessions and cookies to ensure user security.
  • Article Management: Users have the ability to create, edit, and delete their own articles, with an intuitive user interface.
  • Interactive Comments: A comments system was incorporated to allow readers to interact and respond to existing comments.
  • Search and Filters: Users can search for articles by keywords and filter them by categories or tags.
  • Responsive Design: The platform adapts to different devices using Bootstrap, providing a consistent user experience.

This project has enabled me to strengthen my backend and frontend development skills, and I am excited about the future opportunities this interactive blog offers.
